A simple and secure Random Password Generator built using JavaScript.
This project generates strong passwords and allows users to copy them to the clipboard.

The project is deployed on Netlify and integrated into my React portfolio.

# Features
- *Generate random strong passwords*
- *Fixed password length*
- *Includes : Uppercase letters , Lowercase letters , Numbers , Special characters*
- *Responsive and lightweight*

# Teach stack
- *HTML*
- *CSS*
- *JavaScript*
- *Clipboard API*

# Work
- *Click Generate Password*
- *JavaScript randomly selects characters from a predefined character set*
- *Password is displayed in the input field*
- *Password is copied using the navigator.clipboard API*
